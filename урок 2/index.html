HTML

Дата добавления:13.10.2024,Дата обновления:22.11.2024

HTML (от англ. HyperText Markup Language «язык гипертекстовой разметки») — стандартизированный язык гипертекстовой разметки документов для просмотра веб-страниц в браузере.
Веб-браузеры получают HTML документ от сервера по протоколам HTTP/HTTPS или открывают с локального диска, далее интерпретируют код в интерфейс, который будет отображаться на экране монитора.

Жирный-парный тег b делает текст жирным

Курсив-парный тег i делает текст курсивным

Подчёркивание-парный тег u добовляет подчёркивание к тексту

Зачёркивание-парный тег s добовляет зачёркивание к тексту

Маркированый тег-парный тег mark добовляет жёлтый фон

Одиночный тег br переносит текст в новую строчку
